# Cross-Target SHAP Entropy Analysis of Kinase Inhibitor Activity

**A Framework for Quantifying Shared and Target-Specific Structure-Activity Relationships Across Kinase Targets Using Cross-Target SHAP Entropy Analysis**

[![Journal](https://img.shields.io/badge/Journal-Molecular%20Diversity-blue)](https://link.springer.com/article/10.1007/s11030-026-11664-6) [![Python](https://img.shields.io/badge/Python-3.11-green)](https://www.python.org/) [![License](https://img.shields.io/badge/License-MIT-yellow)](https://github.com/Asidsal/Kinase-inhibitor-activity/blob/main/LICENSE)

---

## Overview

This repository contains the complete, reproducible analysis pipeline for:

Salihu, A.S., Rahama, M.S., Wan Salleh, W.M.N.H., Gwaram, N.S., Salisu, A., Yusuf, S.S. (2026) Chemical features consistently associated with kinase inhibitor activity: a cross-target SHAP analysis under scaffold-split evaluation. *Molecular Diversity*. https://doi.org/10.1007/s11030-026-11664-6

The study introduces a cross-target SHAP entropy framework that decomposes Morgan fingerprint feature importance profiles into consensus components (shared across kinase targets) and divergent components (target-specific), applied to 31 human kinase targets from BindingDB under scaffold-based evaluation.

**Key findings:**

- Random Forest classifiers achieved median AUROC 0.994, AUPRC 0.9998, MCC 0.674
- Cross-target SHAP profile consistency was moderate (mean Spearman r = 0.332; 465 target pairs)
- 15 consensus features dominated by aromatic N-heterocycles, aliphatic rings, and hydrogen bond environments
- 50 divergent features with 2.8-fold higher SHAP magnitude than consensus features
- SAR validation confirmed genuine enrichment of top consensus fragments in active compounds
- The entropy decomposition framework is generalisable to other protein families

## Usage

1. Download the data files from Zenodo and place them in a directory
2. Open `article_03_kinase_shap_github.ipynb`
3. In **Cell 1**, set `ART01_DIR` to point to the directory containing the data files:

```python
ART01_DIR = Path("/path/to/your/data/directory")
```

4. Run all cells in order. Total runtime is approximately 35-40 minutes, dominated by Cell 5 (SHAP computation, ~30 min)

> **Note:** Cell 5 (SHAP computation) requires `check_additivity=False` due to a known interaction between RF models with balanced class weights and the SHAP additivity checker in newer versions of the `shap` library. This does not affect the SHAP values themselves.

## Scope statement

All SHAP attributions in this study derive from a single scaffold-based train-test split per target. Stability of feature attributions across alternative splits was not assessed. Interpretations describe chemical patterns associated with model predictions under this specific data partition and should not be understood as split-invariant chemical rules. The cross-target aggregation across 31 independently constructed scaffold splits provides implicit stability evidence at the consensus level.
